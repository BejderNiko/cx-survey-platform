# CX-Survey-Platform
Intern survey-platform til OK CX &amp; Market Insights

Samlet set er der årlige udgifter på +500.000DKK om året

Vi står i en situation, hvor vi bruger X til kun én ting, Y til enkelte udsendelser og Z til andre udsendelser. Alle tools bliver dermed ikke udnyttet til det fulde, men vi betaler den fulde pris for licenserne. Det betyder, at vi med fordel kan samle det hele i én platform – både for at spare penge, men også for at mindske kompleksiteten for brugen af systemer.
